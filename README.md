# Sach — Developer Portfolio

A clean, fully functional personal portfolio website built with pure HTML and CSS. No frameworks, no npm, no build step — just open and it works.

## Features

- Responsive design (mobile + desktop)
- Smooth scroll reveal animations
- Working contact form with validation
- Mobile hamburger menu
- Active nav highlight on scroll
- Back-to-top button
- Project cards with hover effects
- Skills section

## How to Run Locally

Just open `index.html` in your browser — no server needed.

Or use VS Code Live Server for auto-reload while editing.

## How to Deploy on GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, root `/`
4. Your site will be live at `https://yourusername.github.io/repo-name`

## Customisation

Everything is in `index.html`. Search and replace:
- `Sach` → your name
- Project cards → your real projects
- Contact links → your email, GitHub, LinkedIn
- Stat numbers → your real stats

## Tech Stack

- HTML5
- CSS3 (custom properties, grid, flexbox, animations)
- Vanilla JavaScript (scroll reveal, form validation, hamburger menu)
- Google Fonts (Fraunces + DM Sans)
